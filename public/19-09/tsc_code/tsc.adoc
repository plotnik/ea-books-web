== CHAPTER 1. Your First TypeScript Application

=== Creating the Project

.Initializing the Project
----
cd todo
npm init --yes
----

.Listing 1-6. The Contents of the tsconfig.json File in the todo Folder
```json
{
    "compilerOptions": {
        "target": "es2018",
        "outDir": "./dist",
        "rootDir": "./src",
        "module": "commonjs" <1>
    }
}
```

.Compiling and Executing the Code
----
tsc
node dist/index.js
----
